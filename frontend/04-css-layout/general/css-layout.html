Ejemplos educativos de CSS Layout moderno

1. position: static, relative, absolute, fixed, sticky

static
relative
absolute
fixed
sticky
  • static: Posición por defecto, sigue el flujo normal.
  • relative: Se mueve respecto a su posición original.
  • absolute: Se posiciona respecto al ancestro posicionado más cercano.
  • fixed: Se posiciona respecto a la ventana del navegador.
  • sticky: Se comporta como relative hasta que se "pega" al hacer scroll.

2. z-index

z-index: 1
z-index: 3
z-index: 2

Controla el orden de apilamiento de elementos posicionados. Un valor mayor se muestra encima.

3. box-sizing

content-box
border-box

content-box (por defecto): el tamaño solo incluye el contenido.
border-box: el tamaño incluye padding y borde.

4. overflow

visible
hidden
scroll
auto

Controla qué pasa cuando el contenido se desborda del contenedor.

5. margin y padding

margin
padding

margin: espacio fuera del borde.
padding: espacio dentro del borde.

6. aspect-ratio

aspect-ratio: 16/9
aspect-ratio: 1/1

Mantiene la proporción del ancho y alto del elemento.
